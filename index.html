Fourmolu

Current release: v0.12.0.0

Fourmolu is a formatter for Haskell source code. It is a fork of Ormolu, with upstream improvements continually merged. Fourmolu can be seen as a less-opinionated version of Ormolu: Ormolu implements “one true formatting style”, whereas Fourmolu is configurable. Try it out below!
